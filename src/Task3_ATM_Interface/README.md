# ATM Interface

A simple Java console-based ATM application developed as part of the CodSoft Java Development Internship. This project demonstrates Object-Oriented Programming (OOP) concepts by simulating basic ATM operations.

## Features

- Check Account Balance
- Deposit Money
- Withdraw Money
- Input Validation
- Insufficient Balance Check
- Menu-Driven Console Interface
- Object-Oriented Design

## Technologies Used

- Java
- IntelliJ IDEA
- Object-Oriented Programming (OOP)
- Scanner Class

## Project Structure

```

└── README.md
```

## How It Works

1. Start the application.
2. Choose an option from the ATM menu.
3. Deposit or withdraw money.
4. Check the updated account balance.
5. Exit the application.
