# Student Grade Calculator

A simple Java console application that calculates a student's total marks, average percentage, and grade based on marks entered for multiple subjects.

## Features

- Enter marks for any number of subjects
- Input validation (marks must be between 0 and 100)
- Calculates total marks
- Calculates average percentage
- Assigns grade based on percentage
- Displays the final result in a clean format

## Technologies Used

- Java
- IntelliJ IDEA
- Scanner Class
- Loops
- Conditional Statements

## Grade Criteria

| Percentage | Grade |
|------------|-------|
| 90 - 100 | A+ |
| 80 - 89 | A |
| 70 - 79 | B |
| 60 - 69 | C |
| 50 - 59 | D |
| Below 50 | F |
