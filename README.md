## Task 1 - Number Game

### Description
This is a Java Number Guessing Game developed as part of the CodSoft Java Development Internship.

### Features
- Random number generated between 1 and 100
- User guesses the number
- Too High / Too Low hints
- Maximum 5 attempts
- Multiple rounds (Play Again)
- Score tracking

### Technologies Used
- Java
- IntelliJ IDEA
