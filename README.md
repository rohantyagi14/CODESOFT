# CodSoft Java Development Internship

This repository contains the tasks completed during my **CodSoft Java Development Internship**. Each project is developed in Java and demonstrates problem-solving skills, Object-Oriented Programming (OOP), and core Java concepts.

## Tasks Completed

### Task 1 - Number Game
A console-based number guessing game where the user tries to guess a randomly generated number.

**Features**
- Random number generation
- Limited attempts
- Score calculation
- Multiple rounds

### Task 2 - Student Grade Calculator
A Java application that calculates the total marks, average percentage, and grade based on the marks entered by the user.

**Features**
- Total marks calculation
- Average percentage calculation
- Grade assignment
- Input validation

### Task 3 - ATM Interface
A console-based ATM application using Object-Oriented Programming concepts.

**Features**
- Check Balance
- Deposit Money
- Withdraw Money
- Input Validation
- Insufficient Balance Check
- Menu-Driven Interface
